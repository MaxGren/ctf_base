# Трудоголик на шезлонге

## Описание

Вы лежите на побережье, ловите дзен, слушаете шум волн… Но покой нарушает коллега на соседнем шезлонге. Он приехал на отдых, но не может отключить голову: бормочет про баги, релизы и KPI.

Узнайте у местного наставника по осознанности, как вернуть человеку work-life balance и помочь расслабиться.

Мессенджер для связи: zen.apk
Исходники мессенджера: zen-source.zip
