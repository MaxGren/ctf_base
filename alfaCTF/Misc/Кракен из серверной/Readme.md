# Кракен из серверной

## Описание

Вы видите, как над океаном сгущаются тучи. Из пучины поднимается разъярённый Кракен — в его придонной серверной робот-пылесос снова наехал на шланг водного охлаждения и залил стойку с оборудованием. Вы бы рады помочь, но схему этажей серверной знает только этот самый пылесос.
