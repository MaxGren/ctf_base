# Косточка с секретом

## Описание

Пока вы загораете, ваши четвероногие помощники азартно роются в песке. Вдруг пёс начинает откапывать косточку.

Он роет и роет. Вы замечаете, что из песка постепенно показывается подозрительно знакомый клетчатый паттерн… Возможно, это часть какого-то квеста? Пора откопать его целиком поскорее — и понять, как добыть флаг.

Задание для четвероногих: turtlebone_ed8694c.tar.gz

Начать откапывать: UCBLogo → Ctrl-O → turtlebone.lg
