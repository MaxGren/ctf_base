# Краб-фрилансер на грани

## Описание

На пляже вы натыкаетесь на краба, который строчит клешнями по клавиатуре, параллельно созванивается с клиентом… а ещё пишет себе ТЗ, сам его правит и согласовывает. Он давно забыл, что такое выходные: краб на фрилансе, а значит — всегда в онлайне.

Помогите ему наладить процессы: расставить приоритеты, делегировать рутину и вернуть себе хотя бы вечер пятницы. И исполните мечту крабика — отправьте давно забытое им письмо в файле /letter.txt
