# Десант блогеров

## Описание

На остров прибыли новые отдыхающие — блогеры. Их правая рука срослась со смартфоном, они постоянно что-то снимают и восхищаются местными красотами

Блогеры планируют провести масштабный стрим, но этому мешает медленное интернет-соединение. В одном из магазинов есть необходимый роутер, однако он слишком дорогой

Помогите им приобрести высокоскоростной роутер, чтобы о прекрасном острове узнало как можно больше людей
