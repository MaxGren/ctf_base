# Поток флуда

## Описание

Вы решаете немного отдохнуть в тени пальмы и вдруг слышите крики: «Спасайтесь! Потоп!»

Оборачиваетесь и видите, как по пляжу несётся ревущий поток навязчивой рекламы: «Купите!», «Бесплатные фриспины!!!», «Ставки на спорт!»

Скорее отключите рекламу, иначе весь остров накроет волной флуда
