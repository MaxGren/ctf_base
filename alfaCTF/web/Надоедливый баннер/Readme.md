# Надоедливый баннер

## Описание

На горизонте всплывает огромный баннер: «Вам нужен всего лишь простой советский...». О чём именно речь — непонятно. Но реклама преследует вас по всему острову и не даёт наслаждаться видами.

Обычный крестик не работает — ох уж эти чёрные паттерны... или красные? Раскопайте, как отключается назойливая реклама советского магазина, пока всё небо не покрылось всплывающими окнами.
