# Штормовое предупреждение

## Описание

Сегодня непогода — на остров надвигается страшный шторм. Отдыхающие попрятались по бунгало, а вы достаёте бинокль, приглядываетесь к горизонту и понимаете: ситуация не так проста. На самом деле огромная туча — это переполненное облако. Если облачную платформу не почистить, гроза никогда не прекратится. Найдите проект, который всё ломает и освободите место в облаке.
