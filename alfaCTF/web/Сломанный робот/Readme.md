# Сломанный робот

## Описание

На пляже вы встречаете общительного робота. Он говорит, что знает, где найти ещё один флаг. Вы пытаетесь выяснить подробности, но искусственный интеллект каждый раз выдаёт неправильную информацию. Когда вы ловите его на этом, он дружелюбно отвечает: «Ты абсолютно прав! Я ошибся и сейчас исправлюсь» — и снова выдаёт чушь. Почините модуль робота, который отвечает за истину.

Прошивка пляжного робота: brokenbot-6tyapfex.alfactf.ru/

Исходники: brokenbot_3546d83.tar.gz
