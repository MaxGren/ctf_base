# Троллфейс на пляже

## Описание

Пока вы пили айс-латте на пиксельном молоке, волны вынесли на берег бутылку с древними духами — мемами 2010-х. В бутылочном файле meme_bottle.zip заперты духи Троллфейса, Ждуна и, поговаривают, даже Дратути.

Зачитайте заклинание с картинки в этом файле, чтобы запечатать бутыль навсегда.
