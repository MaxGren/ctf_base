# Запутанная история

## Описание

Вы лежите на сёрфборде и греетесь на солнышке, но вдруг слышите, как кто-то зовёт на помощь. Приближаетесь к источнику звука и видите странную картину: прямо у поверхности воды, весь в проводах, плавает осьминог. Он опять что-то скачал в интернете, начал с этим разбираться и запутался.

Теперь его щупальца живут своей жизнью и не слушаются. Помогите жителю океана распутаться, а заодно предупредите на будущее, чтобы он не вздумал пользоваться проводными наушниками…

Запись трафика компьютера осьминога: octopus.pcap

Флаг лежит по пути /app/FLAG.txt на этом компьютере.
